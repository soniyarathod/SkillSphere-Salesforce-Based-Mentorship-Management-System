# SkillSphere-Salesforce-Based-Mentorship-Management-System

SkillSphere is a Salesforce-based Mentorship Management System developed to automate mentor-learner interactions, mentorship requests, session management, learning progress tracking, and analytics reporting.

## Features

- Mentor and Learner Management
- Skill-Based Mentorship Requests
- Automated Approval Workflow
- Automatic Session Creation
- Student Skill Tracking
- Email Notification Automation
- Reports and Dashboards
- Validation Rules and Workflow Automation

## Technologies Used

| Technology | Purpose |
|------------|---------|
| Salesforce CRM | Main CRM Platform |
| Custom Objects | Data Management |
| Lookup Relationships | Object Connectivity |
| Salesforce Flow | Workflow Automation |
| Validation Rules | Data Validation |
| Email Alerts | Automated Notifications |
| Reports | Data Analysis |
| Dashboards | Data Visualization |
| Profiles and Roles | Access Management |

## Workflow

1. Learner submits mentorship request
2. Mentor receives email notification
3. Mentor approves/rejects request
4. Session is automatically created
5. Learning progress is updated
6. Reports and dashboards update dynamically
